Dual-Axis Solar Tracker

Sistem cerdas berbasis IoT dengan Visualisasi Digital Twin 3D Real-Time untuk optimalisasi penyerapan energi matahari.

Mulai Monitoring Pelajari Proyek

Tentang Proyek

Dual-Axis Precision

Menggunakan dua motor servo untuk mengikuti pergerakan matahari secara horizontal (Azimuth) dan vertikal (Elevation).

IoT Integration

ESP32 sebagai otak sistem mengirim data sensor melalui protokol WebSockets untuk latensi rendah.

Real-Time 3D

Visualisasi dome parametrik menggunakan Three.js untuk representasi posisi matahari secara akurat.
